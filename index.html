优势 / Features

轻量化

基于.Net Framework 4.8进行开发，其优越内存自动回收机制使Ph0en1x轻量化。

定制化

Ph0en1x可基于你的需求选择是否启用代理服务(TLSP)，并允许PVP客户端进入你喜爱的服务器。

效率化

相较于WPFLauncher，Ph0en1x大大简化了进入网易服务器的流程，无视网易任何检测，事半功倍。

功能化

我们的特色功能KickPlayers(踢出选定服务器玩家)将内置于Ph0en1x，您只需要使用快捷键即可踢出玩家。

安全化

作为尊重每一位用户隐私的团队，Ph0en1x与服务端均采用AES-CBC-128加密方式进行数据传输。

简洁化

从启动Ph0en1x到进入您喜爱的服务器，您甚至最少只需点按4次。轻快明了，简洁流畅。

开发接口 / API

func/HttpCoder/true

请求模式:POST,Params:{"Header":"...","Body":"..."}

作用:Netease Http执行加密操作

func/HttpCoder/false

请求模式:POST,Params:{"Body":"..."}

作用:Netease Http/Login执行解密操作

func/GetToken/{uniqueID}

请求模式:POST,Params:{"Header":"...","Body":"..."}

作用:Netease Http请求验证令牌

func/GetAuthMessage/{uniqueID}

请求模式:POST,Params:{serverId,gameId,modList

,gameVersion,LaunchWrapperHash,GameDataHash}

作用:获取进服验证报文

func/GetParams/{uniqueID}

请求模式:POST,Params:{username,passwordHash,

deviceId,uniqueId}

作用:获取MpayLogin必要参数
